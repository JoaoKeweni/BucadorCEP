# 📦 Consulta CEP - Java

Projeto simples desenvolvido em Java com o objetivo de consumir uma API pública de CEP e retornar os dados de endereço ao usuário.

## 🚀 Sobre o projeto

Este projeto foi criado como parte de um desafio prático da Alura, com foco em:

- Consumo de API REST
- Manipulação de JSON
- Organização de código em Java
- Separação de responsabilidades

A aplicação realiza uma requisição a uma API de CEP e exibe informações como rua, cidade, estado, entre outros.

## 🛠️ Tecnologias utilizadas

- Java
- API ViaCEP
- Biblioteca para manipulação de JSON (ex: Gson ou similar)

## 📌 Funcionalidades

- Buscar endereço a partir de um CEP
- Exibir os dados formatados no console
- (Opcional) Salvar os dados em arquivo JSON

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram trabalhados conceitos importantes como:

- Requisições HTTP em Java
- Conversão de JSON para objetos Java
- Tratamento básico de erros
- Organização e estruturação de código
